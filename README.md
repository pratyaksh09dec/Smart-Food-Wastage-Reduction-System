# Smart-Food-Wastage-Reduction-System
This project focuses on Food Demand Analysis using Python and data analytics techniques. The system analyzes food demand data, detects missing values and outliers, performs preprocessing, and generates insights related to food preparation, consumption, surplus, and waste percentage.

The project is implemented using:
    Python
    NumPy
    Pandas
    Matplotlib

The notebook performs:
Data cleaning and preprocessing
Outlier detection
Noise and distribution analysis
Feature engineering
Waste percentage calculation
Demand and surplus analysis

Features
1. Data Preprocessing
    Missing value checking
    Data type analysis
    Dataset information extraction
2. Outlier Detection
    Boxplot visualization for detecting anomalies in food orders
3. Noise and Distribution Analysis
    Histogram analysis of order distribution
4. Feature Engineering
    The notebook creates additional useful features such as:
    Meal categories (Breakfast, Lunch, Dinner)
    Prepared quantity
    Consumed quantity
    Food surplus
    Waste percentage
5. Waste Analysis
    Calculates:
    Waste_% = (Surplus / Prepared_Quantity) * 100

Technologies Used
    Python 3.x
    NumPy
    Pandas
    Matplotlib
    Jupyter Notebook
